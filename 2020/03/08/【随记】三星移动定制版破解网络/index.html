【随记】三星移动定制版破解网络

发表于 2020-03-08 | 分类于 Android手机破解相关

简介

关于国行N9108V安卓6.0我参照https://www.cnblogs.com/jsp1256/p/7536579.html三星S5_G9008V 解锁成功，通过解析文章的思路，大概跟早期4.4时期破解思路类似，只不过后期更新关掉了手机端直接编辑相关文件的权限，但是通过这个思路就是用过修改EFS文件来实现破解，而且理论上刷机不会刷写EFS分区，达成永久破解。
实际上早期部分手机，尤其是三星旗舰机支持双网的4g，但是在配置文件内软屏蔽，像现在的三网随意已经没有这种事情了。其实这种事情主要存在于三网来临的前夕，硬件满足但各营运商又想去保护自己的用户不想让丫用其他营运商卡的时代。

分析问题

原文章链接 http://bbs.gfan.com/android-7908992-1-1.html。
纯属转载备份
当然有自己分析了下这个过程，几乎没有风险，使用的是高通官方的工具，修改网络配置文件。

解决问题

1.在拨号键盘输入*#0808#，选择DM+MODEM+ADB模式。（破解完后记得改回去）（小白注：改回去选第一个，MTP即可，第二个MTP+ADB会开了你的USB调试）
2.在我的电脑-》右键-》属性-》设备管理-》端口下记住三星usb链接的com号
若没有说明驱动安装失败。
3.打开高通QPST的连接配置工具QPST comfiguration，并手动添加端口（小白注：就是上面的COM号）
添加完成后，会在QPST Configuration中看到端口处于连接状态添加完成后，会在QPST Configuration中看到端口处于连接状态
4.打开EFS Explorer,选择MSM8974对应的端口，点击ok，等待一会
5.EFS Explorer与手机连接成功后，在左侧目录找到policyman文件夹，在carrier_policy.xml文件那右键先copy date file to pc把文件复制到电脑，再->dalete删除，空白处右键->copy date file from pc
点击browe,选择刚刚下载的carrier_policy.xml，ok

然后重启！
